## Project Overview
- **Name:** Hero Combo Creator
- **Game:** Rise of Castles: Ice & Fire (VTS State 1097)
- **Tech Stack:** Vanilla HTML/JS/CSS (Single Page App), Tailwind CSS (via CDN), Firebase (Firestore for comments/combos), Qwen API for OCR analytics.
- **Key Features:** Manual Builder, Combo Generator, Hero Atlas, Eden Map Planner, Research Calculator, Eden Loyalty Calculator, VTS Admin Dashboard (OCR Analytics).
