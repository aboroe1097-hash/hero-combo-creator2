# Dev Notes

This folder keeps maintainer and AI-agent context out of the repository root.

| File | Purpose |
|------|---------|
| `AGENTS.md` | Project knowledge base and implementation conventions for AI agents. |
| `SESSION_LOG.md` | Historical engineering session notes. |
| `project-memory.md` | Rolling project memory and follow-up ideas. |

These files are not user-facing product documentation. Keep community contribution instructions in `CONTRIBUTING.md`, release history in `CHANGELOG.md`, and release process in `VERSIONING.md`.
